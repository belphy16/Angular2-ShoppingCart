# Angular 4 - Shopping Cart
 
    Shopping Cart application for viewing Catalog and adding shopping items.

## Concepts Covered

   1. String Interpolation
   2. Property Binding
   3. Two Way Binding
   4. @ViewChild
   5. @ContentChild
   6. @Input decorator
   7. @Output decorator
   8. Component Interaction - parent to child and child to parent.
   9. Model creation
   10. Local Reference
