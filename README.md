# Angular 4 - Shopping Cart
 
    Shopping Cart application for viewing Catalog and adding shopping items.

## Concepts Covered

   1. String Interpolation
   2. Property Binding
   3. Two Way Binding
   4. @ViewChild
   5. Dependency Injection
   6. @Input decorator
   7. Services
   8. Component Interaction - through services.
   9. Model creation
   10. Local Reference
